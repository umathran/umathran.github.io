Cows and Bulls (also known as Bulls and Cows) is an old codebreaking mind or paper-pencil game for two or more players. It is a game played with numbers or words that may date back a century or more. Originally, the game was played with a 4-digit number (without repeating digits) or a 4-letter isogram. An isogram is a term for word with non-repeating letters. The rules can be applied on any-digit number or any-letter isogram.

One player, the host, thinks of a number (or word) and tells the numbers of digits (or letters) to the other player(s). Other player(s), the guesser(s), try to figure out the number (or word) by guessing isograms containing same number of digits (or letters). The host responds to the number of cows and bulls for each guess. A 'cow' means a letter in wrong position when compared to the host's word. A 'bull' means a letter in the correct position.

The game ends when either a guesser scores '4 Bulls' or the number of guesses remains is zero.
